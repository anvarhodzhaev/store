# Приложение для управления предложениями поставщиков

React-приложение для оценки прайсов и отправки в WhatsApp-группы.

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

Для запуска в режиме разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## Авторизация

При первом открытии приложения требуется авторизация. Доступ к основному функционалу возможен только после успешного входа.

**Тестовые учетные данные:**
- `admin` / `admin123`
- `user` / `user123`
- `manager` / `manager123`

> ⚠️ **Важно:** В продакшене необходимо заменить простую проверку credentials на реальную авторизацию через API. См. файл `src/components/Login.jsx` для настройки.

Состояние авторизации сохраняется в `localStorage`, поэтому при следующем открытии приложения повторный вход не требуется (до выхода из системы).

## Функциональность

- ✅ **Авторизация** - защита доступа к приложению
- ✅ **Sidebar** - навигация по разделам приложения
- ✅ Отображение лотов поставщиков в виде карточек
- ✅ Фильтрация по статусу и поставщику
- ✅ Переключение между светлой и тёмной темой
- ✅ Настройка интервала автообновления (5, 15, 30, 60 секунд)
- ✅ Отправка/отклонение лотов с указанием маржи
- ✅ Уведомление поставщиков
- ✅ Toast-уведомления для обратной связи
- ✅ Адаптивный дизайн для мобильных устройств

## Технологии

- React 18
- Vite
- CSS Variables для тем
