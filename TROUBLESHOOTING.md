# Решение проблем

Если страница не работает, выполните следующие шаги:

## 1. Установите зависимости

```bash
npm install
```

## 2. Запустите dev сервер

```bash
npm run dev
```

После запуска откройте в браузере адрес, который покажет Vite (обычно `http://localhost:5173`)

## 3. Проверьте консоль браузера

Откройте DevTools (F12) и проверьте вкладку Console на наличие ошибок.

## 4. Проверьте консоль терминала

Убедитесь, что нет ошибок при запуске `npm run dev`

## 5. Очистите кэш

Если проблемы продолжаются:

```bash
# Удалите node_modules и package-lock.json
rm -rf node_modules package-lock.json

# Переустановите зависимости
npm install

# Запустите снова
npm run dev
```

## Возможные проблемы:

1. **Порт занят** - Vite попытается использовать другой порт автоматически
2. **Зависимости не установлены** - выполните `npm install`
3. **Ошибки в коде** - проверьте консоль браузера (F12)
